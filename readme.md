# Slick Review App V1
UI for the Slick Review App V1

## Dependencies
* Npm >= v6
* NodeJs >= v10

## Install
```bash
sudo npm install
```
## Workflow
```bash
# continuous incremental rebuild + web server (dev version)
npm start

# continuous incremental rebuild + web server + proxy ws calls
./node_modules/.bin/gulp start:proxy --proxy 'http://localhost:9000'
```
